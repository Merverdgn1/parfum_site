# Sezer Parfüm – Online Parfüm Sitesi

Bu proje, PHP ve MySQL kullanılarak geliştirilmiş basit bir **online parfüm satış sitesi** uygulamasıdır.  
Kullanıcılar ürünleri görüntüleyebilir, sepetlerine ekleyebilir ve sipariş verebilir.

## Özellikler

- Kullanıcı kayıt ve giriş sistemi
- Ürün listeleme sayfası
- Ürün detayları
- Sepete ürün ekleme / sepeti görüntüleme
- Sipariş tamamlama sayfası
- Admin paneli:
  - Ürün ekleme / silme / güncelleme
  - Kullanıcıları ve siparişleri yönetme (projenin durumuna göre eklenebilir)

## Kullanılan Teknolojiler

- **PHP** (procedural)
- **MySQL** (XAMPP üzerinden)
- **HTML5, CSS3**
- **JavaScript**
- **XAMPP (Apache + MySQL)**
